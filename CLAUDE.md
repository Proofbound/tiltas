# Tiltas Partners — Static Website

**Lithuanian-US Market Entry Consultancy** — tiltausa.com

## Project

Static website for Tiltas Partners, a boutique consultancy helping Lithuanian companies in food, supplements, and health tech launch in the US market.

## Tech

- Static HTML/CSS/JS — no build tools
- Bilingual: Lithuanian primary, English toggle (🇺🇸 flag)
- Uses ProofBound shared design tokens via CDN (`brand.proofbound.com/css/tokens.css`)
- Color palette derived from logo: LT flag green/yellow/red + US navy

## Repo

- **GitHub:** https://github.com/Proofbound/tiltas
- **GitHub Pages:** https://proofbound.github.io/tiltas/
- **Branch:** `master`

## Structure

- `index.html` — single-page site (hero, services, sectors, about, contact)
- `css/style.css` — site styles built on ProofBound shared tokens
- `assets/` — logo SVG and other static assets
